Discuss the scope of var, let, and const ?

Var : var is not a block scope. That's mean if a variable declare in var it can be access anywhere in code. That also mean if var use for declaring a variable it become global scope


Let : let is a block scope. If a variable declare with let Keyword inside a function that it become a block scope. It can not be access outside this function That also mean. If a variable declare with let in global it can be access anywhere in the code also any where inside the function.


Const : Const is a highly strict reserve keyword in javascript for declaring a variable. If you declare it it never in global not inside of function then the variable name must by use as ones. The same variable name not declare again anywhere in global. And it also has a similar behavior like let

Tell us the use cases of null and undefined !
